# LR5-barrier-function-method

Установка зависимостей
```bash
pip install -r requirements.txt
```

Запуск
```bash
python main.py
```

Использование
```bash
1. Задаем начальные параметры:
    1.1. Начальная точка X0
    1.2. Параметр μ
    1.3. Параметр β
    1.4. Критерий остановки ε

2. Задаем параметры для градиентного спуска

3. Нажимаем "Запустить метод"
```

Насчет графика
```bash
График при запуске может странно отображаться.
Для нормализации отображения графика нужно сдвинуть область между областями "Результаты итераций" и "График"
```
